# Changelog

## 0.1.0

- Initial release of `@isvend/expo-udp`.
- Added low-level UDP socket APIs for iOS and Android.
- Added bind, send, receive, close, address, broadcast, multicast, TTL, and loopback controls.
- Added `useUdpSocket` as a thin page-local React lifecycle helper.
- Added Expo config plugin support for local network and multicast permissions.
